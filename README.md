# Projeto-DevOps-01

Neste projeto realizei a conteinerização de um website estático (HTML, CSS e JavaScript) utilizando Docker, aplicando conceitos de criação de imagens, gerenciamento de containers.

A imagem foi versionada e armazenada no Amazon Elastic Container Registry (ECR), permitindo o gerenciamento centralizado.

O deploy foi realizado manualmente em uma instância Amazon EC2 na Amazon Web Services, incluindo configuração do ambiente Linux Ubuntu 24.04, permissões de usuário, execução de containers e validação da aplicação em ambiente de nuvem.

Projeto prático com foco em fundamentos de conteinerização, publicação de imagens, infraestrutura em nuvem e fluxo básico de deploy — aplicando os conhecimentos adquiridos no curso Docker Fundamentals da DIO.

🛠️ Tecnologias utilizadas

    Docker
    AWS EC2
    AWS ECR
    Linux
    CLI 
⚙️ Conceitos aplicados

    Conteinerização de aplicações
    Build de imagens Docker
    Tag e versionamento de imagens
    Push para registry privado
    Deploy manual em VM na nuvem
    Demonstração do projeto em execução
